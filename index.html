HTML
HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It uses tags to define elements like headings, paragraphs, images, links, and more.
CSS
CSS (Cascading Style Sheets) is a stylesheet language used to describe the look and formatting of a web page written in HTML. It controls the layout, colors, fonts, spacing, and responsive design of elements on a webpage.
  • "Cascading" means styles are applied in a hierarchy, allowing overrides and inheritance.
  • CSS makes websites visually appealing and user-friendly.
JS
JavaScript (JS) is a scripting language used to make web pages interactive and dynamic. It runs in the browser and can respond to user actions, update content, validate forms, and more.
  • JavaScript works with HTML and CSS to build modern web apps.
